# Adatbázis editor
Adatbáziskezelőprogram PHP-ban.

### Tartalom
- mySQL.php: az SQL kapcsolathoz Libary.
- lekerdez.php: lekérdezi a adatot.
- beilleszt.php: adatot lehet vele beküldeni.
- index.php: főoldal

 ### Közreműködők
 + BB: mySQL.php, lekerdez.php
 + FV: index.php, beilleszt.php
 + BL: szinek, Readme
 + BN: stilus
